### How do I get set up? ###

* Make sure you have nvm installed, should install node v6.2.1 using it
* Clone this repo
* `cd <repo>`

    `npm install`

    `cd node_modules/moonraker`

    `npm install`

* Probably also need chromedriver installed / in your path

### How to run tests
